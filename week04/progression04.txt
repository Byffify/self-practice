map() → เปลี่ยนค่าทุกตัวใน array จากข้อ 1 ได้ array ใหม่ที่เก็บจำนวนตัวอักษรของแต่ละ string

filter() → เลือกบางค่าที่ตรงเงื่อนไข จากข้อ 2 ได้ array ใหม่ที่มีเฉพาะตัวเลขที่มากกว่า 50

reduce() → รวมทุกค่ามาเป็นค่าเดียว จากข้อ 3 ได้ผลรวมของตัวเลข

find() → หาค่าตัวแรกที่ตรงเงื่อนไข จากข้อ 4 ได้ object สินค้าตัวแรกที่มีราคาเกิน 100 บาท

sort() → เรียงลำดับข้อมูล จากข้อ 5 เรียงลำกับตาม A-Z
